# Environment Variables

## Required

- `APP_EDITION`
- `DATABASE_URL`
- `NEXTAUTH_SECRET`
- `NEXTAUTH_URL`
- `CRON_SECRET`
- `EMAIL_PROVIDER`
- `EMAIL_FROM`

## Required When `EMAIL_PROVIDER=resend`

- `RESEND_API_KEY`

## Required When `EMAIL_PROVIDER=smtp`

- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_SECURE`
- `SMTP_USER`
- `SMTP_PASS`

## Recommended

- `DB_POOL_SIZE`
- `SESSION_MAX_AGE_HOURS`
- `SESSION_IDLE_TIMEOUT_HOURS`
- `SESSION_ACTIVITY_THROTTLE_MINUTES`
- `PRISMA_QUERY_LOGS`

## Optional Bootstrap

- `TEST_ADMIN_BOOTSTRAP_ENABLED`
- `TEST_ADMIN_EMAIL`
- `TEST_ADMIN_INITIAL_PASSWORD`
- `TEST_ADMIN_ORG_SLUG`
